Criação de listas

Listas ordenadas

A criação de listas pode ser feita pela tag "ol", no qual delimitará a lista. Já a tag "li" irá identificar cada item da lista. Desta maneira:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

A tag que identificará as listas, usada pela tag "li" não tem obrigatoriedade de fechamento.

Listas não ordenadas

Se caso a ordem dos itens não seja importante, temos a opção das listas não ordenadas, marcada pela tag "ul".

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Misturando as coisas

Podemos criar também listas dentro das listas. Veja

  1. Teoria Cognitivo Comportamental
  2. Psicanálise
    1. Freud
    2. Lacan
    3. Melanie Klein
  3. Humanismo
    1. Carl Rogers
    2. Fritz Perls (gestalt)
