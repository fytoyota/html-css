Semântica e recursos para escrita

Abreviações

Aprenderemos aqui algumas das tags necessárias para marcar o texto de alguma forma ou torná-lo com algum significado, que foi a principal razão da mudança do HTML.


É só passar o mouse por cima da palavra sublinhada e verá a mágica.
Tal recurso é utilizado pela sigla "abbr".


Strong

Aqui utilizaremos outro recurso muito utilizado: o Negrito. Repare que a palavra "negrito" ficou mais escura, mais marcada. Para tal recurso, utiliza-se a tag "strong".


Itálico

Para palavras em Itálico são representadas pela tag "em", de emphasis.


Mark/marca textos

Quando queremos DESTACAR uma frase, uma palavra bem como fazíamos com o marca textos, utilizamos a tag "mark".


Small

Algumas tags se foram, outras ficaram, e uma delas é a tag "small" que diminui o tamanho da frase/palavra.


Texto deletado

Quando queremos riscar uma frase para indicar que aquela determinada frase deve ser desconsiderada. A sigla que utilizamos para tal, chama-se "strike".


Texto inserido

Já quando queremos indicar algum texto que foi inserido ou até mesmo destacá-lo, utilizamos a tag "ins" sublinhar o texto.


Textos sub e subrescritos

Utilizamos quando queremos elevar algum número ao quadrado ou quando iremos trabalhar com a Tabela Periódica.
X2, representado pela sigla "sup" e H2O, representado pela sigla "sub".


Citações

Quando fazemos trabalhos acadêmicos ou científicos, geralmente encontram-se diversas citações durante a leitura. Isso porque a citação diz exatamente aquilo que o autor está querendo representar, mas encontrou uma melhor forma de expressar pela ótica de outro autor. Para citações que não necessitam de deslocamento, utilizamos a tag "q".


Já "blackquote" serve para quando queremos fazer a citação de acordo com as normas ABNT.
.


Endereço

Como o HTML5 veio para dar semântica e, por isso, significado às tags, quando queremos escrever um endereço, então colocamos a tag "address". Por exemplo:

Alameda Pará, 387, Porto Novo - Caraguatatuba/SP
