Código-fonte

A máquina não sublinha as palavras uniformemente, isto é, quando vamos sublinhar alguma palavra/frase em qualquer situação utilizando o computador cada letra tem um "tamanho" diferente. Por exemplo, o comando document.getElementById('teste') é escrito em JavaScript. Se sublinharmos o elemento "t" de "Element" veremos que este tem um tamanho diferente da letra "g". Para isso, utilizamos o código code.

Quando inserimos tal código, então fica desta maneira: document.getElementById('teste')
