Testando imagens e carga de imagens

Abaixo colocarei o arquivo de imagem

Arquivos JPEG permite que seja mais compacto, já PNG é um tipo de arquivo que veio para substituir o GIF, permitindo transparência. Sendo assim, o uso de uma ou outra vai depender de como e para que será utilizada a imagem.
PNG são aqueles logos e outras imagens que tem o fundo transparente, permite que apenas a imagem "principal" se destaque. JPEG são os arquivos de imagem comum.
