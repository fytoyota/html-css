Vamos criar um link

Link que abrirá na mesma página

Para criar um hyperlink, devemos criar âncoras através da tag "a". O principal atributo dessa tag é o "href", que cria uma referência hipertexto.

Continuar navegando no site utilizando a sigla "_self"

Neste caso, criamos uma forma de acessar o link desejado que continuará no mesmo navegador.

href
A sigla siginifica que temos que colocar ali o link, a URL que desejamos ir ao clicarmos.
target
Target é o atributo da tag, ou seja, para onde iremos mandar o link: se continuaremos no mesmo site ou se iremos dar o comando para ir para outra página/aba.
hreflang
Este último indica para a máquina qual o idioma está sendo utilizado naquele link, facilitando que o sistema entenda e possibilite traduções.
rel
Rel é um atributo que ajuda os mecanismos de busca e de guiança do Google, por exemplo, a entenderem o código e para onde o link está mandando.

Link que abrirá outra página

Já quando desejamos guiar o usuário para um link que não continue na mesma página, assim como acontece no Google, por exemplo. Manteremos a tag e alteraremos o atributo, que será "_blank".

Link que guiará para outra aba.

A estrutura do código é semelhante, alterando somente o "target", que é o comando que irá guiar para onde o link deve abrir.


Links internos

Os links internos farão a relação com outra página que está dentro do nosso servidor

Esta é a primeira página do meu site, onde aprenderemos como levá-la a segunda página que está no mesmo servidor.

Por meio deste link, guiaremos você até a segunda página que irá conter um conteúdo para download, clicando aqui.

No caso de links internos, não é necessário o atributo "target", pois neste caso o link é do mesmo servidor, buscará na mesma pasta.

Links para propagandas dos quais você não quer que o mecanismo de busca "relacione seu site com o da pessoa", utilizando o atributo "rel".
