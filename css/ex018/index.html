Testando fontes do Google

Fonte número 1

Neste primeiro teste, utilizamos três fontes diferentes tiradas do Google Fonts. Para tal recurso, o passo a passo se define da seguinte maneira:

  1. Ainda no site do Google Fonts

    2. Selecionaremos a fonte desejada e clicamos em "Get font". Em seguida, ele irá adicionar a "sacola" e ele irá armazenar quantas fontes você selecionar, agrupando todas num código só. No caso, selecionei três tipos de fonte, a Cutive Mono, Patrick Hand SC e a Maiden Orange; as três estão na regra abaixo.

  2. Colocando a regra ao código CSS

    3. Após agrupar todas as fontes desejáveis, o próximo passo é clicar em "Get embed code" e "import". A primeira aba é a regra que devemos colocar logo abaixo do "style" ou nas primeiras linhas do código CSS, ela é quem dirá à máquina para reconhecer tal código. Toda regra é ditada pelo arrouba.

  3. Aplicando no código

    4. Para isso, basta que copiemos o código que fica na barra logo abaixo, ou se preferir, gerar à sua maneira, com as suas configurações de tamanho, estilo, peso etc.

            Assim ficou a regra:
            
                @import url('https://fonts.googleapis.com/css2?family=Cutive+Mono&family=Maiden+Orange&family=Pa
                trick+Hand+SC&display=swap');
            
            E assim ficou o código:
            
                h1{
                    font-family: 'Maiden Orange', serif;
                    text-align: center;
                }
                h2{
                    font-family: 'Patrick Hand SC', serif;
                    font-style: italic;
                }
                p{
                    font-family: 'Cutive Mono', serif;
                    font-weight: 600;
                    font-size: large;
                    text-align: justify;
                }

Testando baixadas de outros sites

Neste caso, usaremos fontes de outros sites que não do Google, o que exige alguns passos a mais, pois o arquivo da fonte estará armazenado numa pasta.

  1. Acesse o site

    2. Acesse os sites que disponibilizam fontes para baixar, como por exemplo o Dafont (site utilizado aqui).

  2. Escolha sua fonte e baixe

    3. Após a escolha da fonte, faça o download. Ele irá baixar um arquivo ZIP, portanto descompacte-o e coloque os arquivos numa pasta de seu HTML/CSS.

  3. Colocando a fonte no seu site

    4. Pela regra "@font-face" colocaremos todas as informações necessárias para que a fonte apareça de maneira correta em seu código.

Esta foi a regra: 

                @font-face {
                font-family: 'Gothic';
                src: url('fonts/MetalGothic-DemoVersion-Regular.ttf') format('truetype');
                font-weight: normal;
                        }
                
            E este foi o código:
                
                    h1{
                        text-align: center;
                        font-family: Gothic, serif;
                    }
