Id e Class

Aprendendo sobre id

O id tem como função identificar uma tag específica para designar características específicas dentro do CSS. Se, por exemplo, tivermos vários H1 e designamos diversas características para tal, mas queremos que um H1 em específico tenha outra(s) características que as demais não possuem, então dentro deste H1 específico, colocamos, dentro da própria tag o "class".Outra característica deste elemento, é que não podemos colocá-lo em mais de uma tag.
