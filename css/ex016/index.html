Testando degradê de cores

É possível colocar as cores de fundo como degradê, o que gera esse efeito visual visto na tela. Esta função das CSS's é tido e entendido pela máquina como uma imagem, uma foto e ele irá ler como sendo linear (de cima pra baixo, baixo pra cima, esquerda pra direita, direita pra esquerda) ou radial (formato em círculo). Assim como na foto, é esperado pela máquina que coloquemos os valores, dimensões etc.
