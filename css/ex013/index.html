CSS's Inline

Quando falamos de CSS, falamos da parte estética do site, já que este código mexe com os estilos, criando cores, tamanhos, formas... enfim, dando o toque visual ao site. Para isso, existem formas diferentes de se montar um código CSS, se é assim que é o correto a dizer. Neste código, utilizaremos a forma Inline, que é basicamente o código escrito linha por linha, tag por tag, parâmetro por parâmetro e por aí vai.

Este código em questão está sendo programado linha por linha, código por código
